# Sales Performance Dashboard (Excel)

## Project Overview
This project demonstrates a sales performance dashboard built using Microsoft Excel
to analyze sales trends by product category, region, and month.

## Objectives
- Identify top-selling products
- Analyze revenue contribution by region
- Track monthly sales trends
- Determine overall sales growth or decline

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Dataset
- Public sales dataset (CSV format)
- Cleaned and structured in Excel

## Key Features
- Interactive dashboard with KPIs
- Product-wise, region-wise, and monthly sales analysis
- Dynamic filtering using Region and Year slicers

## KPIs
- Total Sales
- Total Quantity Sold
- Top Product Category
- Top Revenue Region

## Insights
- Electronics is the top-selling product category
- West region generates the highest revenue
- Sales show seasonal fluctuations with an overall trend

## How to Use
1. Download the Excel file
2. Open `Sales_Performance_Dashboard.xlsx`
3. Use slicers to filter by Region and Year
